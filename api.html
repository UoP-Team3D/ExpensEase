utilities.db_connection.get_db_connection()[source]

Establishes a connection to the database using the DATABASE_URL environment variable.

The DATABASE_URL should be in the format: ‘postgresql://username:password@host:port/dbname’. If the port is not specified, the default PostgreSQL port 5432 is used. Typically it will just be 5432 unless you specified otherwise

Raises:

  • EnvironmentError – If the DATABASE_URL is not set, cannot be parsed,

  • ValueError – If the DATABASE_URL format is invalid

  • ConnectionError – If database connection failed

  • RuntimeError – Any other unexpected issue occurs

Returns:

A connection object to the PostgreSQL database.

Return type:

psycopg2.connection
